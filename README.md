# My Personal Website

My personal portfolio website - I'm Yousuf Alshaaili, an AI & Computer Science student at Sultan Qaboos University.

## Tech
- HTML, CSS (with custom animations)
- JavaScript
- PHP (`sendmail.php`) for the contact form

## Structure
| File | Purpose |
|------|---------|
| `index.html` | Main landing page |
| `styles.css` / `animations.css` | Styling and animations |
| `main.js` | Client-side behavior |
| `sendmail.php` | Contact-form mail handler |
| `sitemap.xml` | Sitemap for search engines |

## Running locally
Open `index.html` in a browser. The contact form (`sendmail.php`) requires a PHP-capable server.
